Nexa Solutions

Projeto Financeiro - Dashboard e Análise de Dados

Sobre o Projeto

O projeto “Nexa Solutions” analisa receitas, despesas e lucros mensais de uma empresa fictícia, gerando insights estratégicos para decisões financeiras.

Os dados foram tratados com Python (Pandas, Numpy) e visualizados em dashboard interativo no Power BI.

Ferramentas Utilizadas

- Python (Pandas, Numpy, Matplotlib, Seaborn)

- Excel/WPS para manipulação de dados

- Power BI para criação do dashboard

Principais Insights

- Margem média de lucro: 25%

- Categoria “Salários” representa aproximadamente 40% dos custos

- Ajustes em despesas fixas e variáveis podem aumentar o lucro líquido em até 10%
